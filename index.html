COLOR WORLD

RGBA Colors

RGBA color values are an extension of RGB color values with an Alpha channel An RGBA color value is specified with:

rgba(red, green, blue, alpha);

Here are some examples:

rgba(99,255,99,0.1)

rgba(99,255,99, 0.2)

rgba(99,255,99, 0.4)

rgba(99,255,99, 0.6)

rgba(99,255,99, 0.8)

rgba(99,255,99, 1)

Alpha's most unique feature is that lower alpha value color looks cool!! if your background is white :)

Do try with white background!!

hsl(hue, saturation, lightness)

Some examples of HSL colors:

hsl(0, 100%, 50%)

hsl(280, 10%, 50%)

hsl(147, 50%, 47%)

hsl(321, 76%, 79%)

hsl(356, 100%, 50%)

hsl(208, 53%, 58%)



HSLA colors


HSLA color values are extension of HSL color values with an Alpha channel

An HSLA color value is specified with:

hsla(hue, saturation, lightness, alpha)

Examples of HSLA colors:

hsla(9, 100%, 68%, 0)

hsla(9, 97%, 68%, 0.2)

hsla(9, 100%, 63%, 0.4)

hsla(9, 99%, 68%, 0.6)

hsla(9, 93%, 69%, 0.8)

hsla(9, 98%, 74%, 1)
